## Installation

1. Copy all files to your GeoFun repository
2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the development server:
   ```bash
   npm start
   ```

4. Build for production:
   ```bash
   npm run build
   ```

## Using as Android App

This app is built as a Progressive Web App (PWA) and can be installed on Android devices:

1. Deploy the built app to a web server (GitHub Pages, Netlify, Vercel, etc.)
2. Open the website on an Android device
3. Tap the menu and select "Add to Home Screen" or "Install App"
4. The app will install like a native Android app
